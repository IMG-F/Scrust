# Pen Blocks

To use these blocks, enable the "pen" extension in your `scrust.toml`: `extensions = ["pen"]`.

| Scrust Syntax | Scratch Block | Notes |
| :--- | :--- | :--- |
| `pen_clear()` | <pre class="blocks">erase all</pre> | |
| `pen_stamp()` | <pre class="blocks">stamp</pre> | |
| `pen_down()` | <pre class="blocks">pen down</pre> | |
| `pen_up()` | <pre class="blocks">pen up</pre> | |
| `set_pen_color(color)` | <pre class="blocks">set pen color to (color)</pre> | |
| `change_pen_hue_by(delta)` | <pre class="blocks">change pen color by (delta)</pre> | Actually changes hue |
| `set_pen_hue_to(value)` | <pre class="blocks">set pen color to (value)</pre> | Actually sets hue |
| `change_pen_shade_by(delta)` | <pre class="blocks">change pen shade by (delta)</pre> | |
| `set_pen_shade_to(value)` | <pre class="blocks">set pen shade to (value)</pre> | |
| `change_pen_size_by(delta)` | <pre class="blocks">change pen size by (delta)</pre> | |
| `set_pen_size_to(size)` | <pre class="blocks">set pen size to (size)</pre> | |
